📌 Project Overview
As mobile internet becomes the primary gateway to the digital world, understanding usage patterns is essential for telecommunication providers and policymakers. This project performs an in-depth analysis of a mobile internet usage dataset to identify trends, user demographics, and behavioral drivers.

The primary goal is to use descriptive and inferential statistics to answer key questions about how different segments of the population consume mobile data.

📊 Key Features
Data Cleaning & Preprocessing: Handling missing values and ensuring data types are optimized for analysis.

Exploratory Data Analysis (EDA): Visualizing the distribution of data usage across different user categories.

Statistical Testing: Applying hypothesis testing to determine if observed differences in usage (e.g., between age groups or genders) are statistically significant.

Correlation Discovery: Identifying relationships between variables like plan type, location, and total data consumption.

🛠️ Tech Stack
Language: Python

Libraries: * Pandas & NumPy: For core data manipulation.

Seaborn & Matplotlib: For generating insightful visualizations (histograms, boxplots, scatter plots).

Scipy.stats: For performing rigorous statistical tests.
